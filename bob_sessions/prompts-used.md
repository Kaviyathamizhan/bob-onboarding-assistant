# Bob Prompts Used — Flask Repo Analysis

## Prompt #1 — Architecture Analysis
```
Here is the file structure and content of a GitHub repository (pallets/flask).
Analyze it and provide:
1) The detected tech stack (languages, frameworks, key libraries)
2) A 2–3 paragraph architecture overview explaining how the system is structured
3) A list of the main modules/services with a brief description of each
```

---

## Prompt #2 — Module Descriptions
```
For each module in the flask codebase, describe:
- Its purpose in the overall system
- Its key files
- What it imports from other modules
- What functions or classes it exports
```

## Prompt #3 — Onboarding Guide
```
Generate a complete onboarding guide for a new developer joining the pallets/flask project.
Include:
1. Where to start reading code
2. Key concepts to understand first
3. The most important files and what they do
4. Codebase conventions and patterns used throughout
```
